This folder aims to utilize pdf files to get gene annotations from Pubtator.

Step1: extract_text_from_pdf_then_xml.py
       Process pdf files to txt files, then to BioC xml files. -- Ready to submit to Pubtator
       
Step 2: SubmitText_request.py
        Submit BioC xml file to Pubtator.
        Please read Section C in Pubtator_readme.txt
        
Step 3: SubmitText_retrieve.py
        Retrieve gene annotations from Pubtator.
        Please read Section C in Pubtator_readme.txt
        
Step 4: get_gene_annotation_from_pubtator_result.py
        Extract gene annotations from Pubtator retrieved file.
